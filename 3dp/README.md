# 3D Printed Case Guide — *Lichee-Jack*

This folder contains all the **3D printable files** for the **Lichee-Jack** case.

Designed for durability and easy assembly using **M2 heat-set inserts** and **screws**.  
Includes optional **RJ45 dummy plug** (for keychain use) and a **60×20 mm badge** plate.

## Step 1 — Print the Case

![](../doc/CASE_WITHSW.jpeg)

Supported materials: **PLA+, PETG, or ABS**

**Recommended print settings:**
- Layer height: 0.16 – 0.20 mm  
- Infill: 30 – 50 %  
- Supports: Only where needed  
- Orientation: Flat face down  

> [!NOTE]  
> Avoid **resin or SLA printing** — these materials are brittle and not suitable for heat-set inserts.

---

## Step 2 — Install Heat-Set Inserts

1. Heat your soldering iron to **180 – 200 °C**.  
2. Align the insert with the hole.  
3. Gently press while heating until it sits flush.  
4. Let cool completely before adding screws.

![](../doc/IMG_E6636.JPG)
![](../doc/IMG_E6637.JPG)

> Tip: Practice on scrap material if this is your first time using heat-set inserts.

---

## Step 3 — Add Fiber to LED Holes

- Insert **plastic optic fiber** into the power LED holes for a diffused glow effect.  
- Trim the ends flush with the surface.

## Step 4 — Final Assembly

1. Align the top and bottom shells.  
2. Place the **switch header** in the side slot before closing.  
3. Secure using **M2 screws**.  
4. Do **not** overtighten — just snug fit.
